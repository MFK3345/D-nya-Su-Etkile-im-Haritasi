# 🌊 Dünya Su Kaynakları İnteraktif Haritası

## 📌 Proje Özeti
Dünya genelindeki tatlı su kaynaklarını görselleştiren, 
World Bank, Wikipedia ve FAO verilerini kullanan 
interaktif 2D/3D harita uygulaması.

## 🎯 Amaç
- 260+ ülkenin su kaynaklarını analiz etmek
- Su kıtlığı risklerini görselleştirmek
- Sürdürülebilir su yönetimi için farkındalık

## 🛠️ Teknolojiler
- **Frontend:** HTML5, CSS3, JavaScript
- **2D Harita:** Leaflet.js
- **3D Harita:** Cesium.js
- **Veri Kaynakları:** 
  - World Bank API (2015-2025 verileri)
  - REST Countries API
  - Wikipedia API (Türkçe)
  - FAO AQUASTAT

## 📊 Özellikler
✅ 260+ ülke ve bölge
✅ Gerçek zamanlı API veri çekme
✅ 2D/3D harita görünümleri
✅ Dinamik renklendirme (su skoru 0-10)
✅ Türkçe arayüz
✅ Detaylı ülke analizleri
✅ Trend grafikleri (2016-2025)
